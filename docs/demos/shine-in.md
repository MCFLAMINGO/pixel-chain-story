# Shine in — non-tech front door

**For:** someone who found Pixel and thinks “that would help my site.”  
**Not for:** operators wiring DNS/rsync (that’s `/continuity` backstage).

## Story

1. Look up Continuity — keep customers when the host blinks.  
2. Finish your website / brand however you already do.  
3. Press **Shine in**.  
4. You’re in the light — no DNS homework.

## Run

```bash
bun run dev
```

Open [http://localhost:8080/shine](http://localhost:8080/shine)

Also linked from the live field (`/` → **Shine in**) and Lab.

Shortcuts:

- **Try with McFlamingo demo** — uses `/mcflamingo/` menu HTML  
- After success → Continuity desk for till / chaos drill (operator tools)

## Honesty

Lab: Continuity record stays in this browser (`localStorage`).  
Not public DNS failover. Not “Pixel replaces AWS compute.”
